Functionality
-------------

painless creates django backend projects without developer coding

it can generate these stuff:

- models.py
- admin.py
- serializers.py
- views.py
- urls.py
- tests
- api documentation
- Dockerfile
- docker-compose.yml

it also can config pro stuff in django:

- Redis cache
- RabbitMQ
